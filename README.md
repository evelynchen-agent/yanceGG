# OpenAI Client Example

This repository provides a minimal Python example that demonstrates how to call the OpenAI API. The API key is intentionally configured with a placeholder value of `"xxxx"` so you can replace it with a real key when deploying.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Usage

```bash
python openai_client.py
```
